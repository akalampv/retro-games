Object of the game

Correctly repeat a longer and longer sequence of signals.

1. Simon will give the first signal. Repeat the signal by pressing the same color lens.

2. Simon will add one signal. Repeat these two signals by pressing the same color lenses, in order.

3. Continue playing as long as you can repeat each sequence of signals correctly.
